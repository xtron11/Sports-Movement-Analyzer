🏋️ Sport Movement Analyzer

AI система анализа техники приседаний в реальном времени на основе компьютерного зрения.

🎯 Что делает проект

Загружаешь видео с приседаниями система автоматически:
- Считает количество повторений
- Анализирует глубину приседа (Perfect / Too Deep / Partial)
- Измеряет наклон спины в градусах
- Генерирует текстовый отчёт по тренировке

🛠️ Стек

- MediaPipe — детекция 33 точек скелета в реальном времени
- OpenCV — обработка видео покадрово
- FastAPI — REST API для интеграции с внешними приложениями
- NumPy — векторные вычисления углов и расстояний
- Loguru — структурированное логирование

📐 Как работает анализ

Система решает нетривиальные задачи компьютерного зрения:

Автодетекция вида съёмки — алгоритм сравнивает длину бедра с шириной плеч и анализирует разницу видимости landmarks, чтобы определить снято ли видео сбоку или спереди — без метаданных и ручных меток.

Нормализация по масштабу — все метрики считаются относительно длины сегментов тела (голень, бедро), а не в абсолютных пикселях. Это делает анализ инвариантным к расстоянию от камеры.

Время через кадры — вместо системного `time.time()` используется `frame_idx / fps`, что даёт точное время внутри видео независимо от нагрузки на CPU при обработке.

Угол наклона спины — вычисляется через `arctan2` вектора плечо→бедро относительно вертикальной оси, а не через стандартный угол между тремя точками.

📊 Метрики качества повторения

| Статус | Угол колена | Описание |
|--------|-------------|----------|
| Perfect | 90–100° | Идеальная глубина |
| Too Deep | < 90° | Слишком глубоко |
| Partial | > 100° | Неполный присед |

Наклон спины считается ошибкой при отклонении от вертикали более 50°.
